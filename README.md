# IMGPROC - 图像处理工具 (静态版本)

## 项目简介

IMGPROC 是一个纯前端的图像处理工具，基于 HTML5 Canvas 实现，无需后端服务即可在浏览器中完成各种图像处理操作。

## 功能特性

### 基础操作
- 旋转（左旋90°、右旋90°）
- 水平翻转
- 垂直翻转
- 裁剪（自定义区域）

### 像素画转换
- 可调节像素尺寸
- 可调节颜色数量
- 输出缩放选项
- 增强模式
- 保持宽高比选项

### 画质增强
- 锐化强度调节
- 对比度调节
- 饱和度调节
- 去噪处理
- 放大倍数设置

### 实用工具
- 水印添加（支持自定义文字、位置和透明度）
- 格式转换（PNG、JPEG、WebP）
- 压缩功能（可调节质量）

## 如何使用

1. **上传图片**：点击或拖放图片到上传区域
2. **选择功能**：在左侧侧边栏选择需要的功能
3. **调整参数**：根据需要调整相关参数
4. **执行操作**：点击对应功能的执行按钮
5. **下载结果**：在底部选择导出格式并点击下载按钮

## 技术实现

- **前端框架**：纯 HTML5 + CSS3 + JavaScript
- **图像处理**：HTML5 Canvas API
- **样式设计**：工业实用风格，深色主题
- **响应式布局**：支持不同屏幕尺寸

## 文件结构

```
ImageProcedure-static/
├── index.html          # 主页面
├── styles.css          # 样式文件
└── script.js           # 核心逻辑
```

## 浏览器兼容性

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 注意事项

- 所有处理均在浏览器本地完成，不会上传图片到任何服务器
- 处理大图片可能会导致浏览器性能下降
- 支持的图片格式：PNG、JPG、BMP、GIF、WebP

## 本地运行

直接在浏览器中打开 `index.html` 文件即可使用，无需安装任何依赖。
